### V) AI and Data Science Skillset Snapshot:

**1. Mathematical Foundations:**

(i) Overall - Data formats, Features (independent variables) and labels/targets (dependent variables), Continuous, discrete, and categorical data

(ii) Linear Algebra: Vectors & Matrices, Matrix Operations, Linear Independence, Vector Spaces, Orthogonality, Eigenvalues & Eigenvectors, Singular Value Decomposition (SVD), Matrix Factorization, Principal Component Analysis (PCA), Vector & Matrix Norms (L1/L2), Cosine Similarity, Least Squares Optimization.

(iii) Probability and Statistics:

Statistical Foundations: Population and Sample, Random Variables (Discrete & Continuous), Probability Distributions, Sampling Techniques (Random Sampling, Stratified Sampling), Sampling Bias, Law of Large Numbers

Descriptive Statistics: Measures of Central Tendency (Mean, Median, Mode), Measures of Dispersion (Variance, Standard Deviation, Range, Interquartile Range), Distribution Analysis (Skewness, Kurtosis), Percentiles & Quartiles, Correlation and Covariance, Exploratory Data Analysis (EDA), Outlier Detection

Inferential Statistics: Probability Theory & Probability Distributions, Central Limit Theorem, Sampling Techniques & Sampling Distributions, Confidence Intervals & Point Estimation, Hypothesis Testing and Statistical Significance, p-value Interpretation

Statistical Tests & Experimentation: Z-Test and t-Test, Chi-Square Test, ANOVA (Analysis of Variance), A/B Testing, Statistical Power Analysis

(iv) Differential Calculus: Limits & Continuity, Derivatives, Higher-Order Derivatives, Partial Derivatives, Chain Rule, Multivariable Differentiation, Gradient & Directional Derivatives, Jacobian and Hessian Matrices, Taylor Series Approximation, Implicit Differentiation, Sensitivity Analysis.

**2. Exploratory Data Analysis (EDA), Data Pre-Processing, Feature Engineering, Feature Selection and Extraction Technique, Dimensinality Reduction**

(i) Exploratory Data Analysis (EDA): Dataset Structure Analysis, data types & schema, summary statistics and descriptive analysis, univariate/bivariate/multivariate analysis, correlation & covariance analysis, data distribution analysis (skewness, kurtosis), outlier detection & anomaly identification, data imbalance analysis, pattern discovery & trend analysis, data visualization techniques (histograms, box plots, scatter plots, heatmaps).

(ii) Data Pre-Processing: Data cleaning and preparation, handling missing values (removal, mean/median/mode imputation, model-based imputation), handling outliers and noisy data, removing duplicates and inconsistent records, data normalization & standardization, encoding categorical variables (Label Encoding, One-Hot Encoding, Target Encoding), data transformation (log transformation, scaling), handling imbalanced datasets (SMOTE, SMOTEENN, undersampling, oversampling), train-test split and cross-validation preparation.

(iii) Feature Engineering: Feature creation from existing variables, domain-driven feature generation, interaction and polynomial features, aggregation-based feature creation, time-based feature extraction (year, month, day, seasonality), text feature engineering (n-grams, TF-IDF features), feature transformation and binning.

(iv) Feature Selection and Extraction Techniques: Filter methods (correlation, chi-square test, mutual information), wrapper methods (Recursive Feature Elimination – RFE), embedded methods (Lasso, Ridge, Elastic Net), tree-based feature importance, variance threshold feature selection.

(vi) Dimensionality Reduction Techniques: Principal Component Analysis (PCA), Linear Discriminant Analysis (LDA)

**3. Machine Learning:**

(i) Supervised Learning: Regression and Classification (Linear Models, Trees, Ensembles, SVM, KNN)

(ii) Unsupervised Learning: Clustering, Association Rules, Anomaly Detection

(iii) Model Training & Validation: Train-test split strategies, bias-variance trade-off analysis, Cross-Validation Techniques, Hyperparameter Tuning model performance comparison

(iv) Model Evaluation Metrics: Classification Metrics (Accuracy, Precision, Recall, F1-Score, Confusion Matrix), Regression Metrics (Mean Absolute Error (MAE), Mean Squared Error (MSE), Root Mean Squared Error (RMSE), R-Squared (R²), Adjusted R²), Ranking & Probabilistic Metrics (ROC Curve, ROC-AUC, Precision–Recall Curve, Log Loss)

  (i) Neural Network Fundamentals - Artificial Neural Networks (ANN), Perceptron and multi-layer perceptron (MLP), Forward Propagation and Backpropagation, Activation Functions, Weight Initialization Techniques, Vanishing and Exploding Gradient Problems, Batch Normalization, Layer Normalization.
  
  (ii) Deep Learning Architectures - Feedforward Neural Networks, CNN, RNN, Transformers
  
  (iii) Loss Functions: Regression Loss Functions, Classification Loss Functions

  (v) Regularization Techniques: L1 and L2 Regularization, Dropout, Early Stopping
